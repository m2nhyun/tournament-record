## Recent Updates

### 2026-03-10 경기 확인 UX 정리 완료
- `submitted`는 `기록됨`, `disputed`는 `재검토 필요` 중심 카피로 정리
- 경기 상세에서 확인 대상 닉네임과 현재 확인 상태를 함께 안내
- `disputed` 경기 상세와 수정 화면에서 `수정 후 다시 저장 -> 새 확인 요청 전송` 흐름을 명시
- 재검토 상태 경기 수정 성공 메시지를 재확인 요청 중심으로 유지

### 2026-03-11 최적화/UI·UX 보강
- 클럽 상세 mutation 후 전체 재조회 대신 로컬 상태 갱신으로 반응성을 개선
- 클럽 상세에 `코드 복사`, `링크 복사`, `링크 공유`, `재발급(owner)` 액션을 역할별로 정리
- `/join/[inviteCode]` 라우트를 임시 비활성화 카드에서 실제 `InviteJoinView`로 다시 연결
- 홈/초대 참가/email 인증 버튼을 모바일 우선 배치로 조정하고 게스트/정회원 분기를 더 명시
- 히스토리 목록 remount를 제거하고 상대 검색을 `내 상대` 기준으로 정밀화
